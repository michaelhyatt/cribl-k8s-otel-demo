# Cribl Lake datasets for Otel traffic
Create 3 Cribl Lake datasets to receive the OTel data:
* `otel_traces`
* `otel_metrics`
* `otel_logs`

These instructions are also mentioned in the [Stream setup](../stream/STREAM_SETUP.md).

# for k8s :

 * k8s_logs
 * k8s_metrics
